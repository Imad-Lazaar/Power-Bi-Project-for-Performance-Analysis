# Power-Bi-Project-for-Performance-Analysis

# Analyse de Performance Financière : YTD vs PYTD

Ce projet consiste en la création d'un tableau de bord interactif sous **Power BI** pour analyser la performance commerciale et comparer les résultats actuels avec l'année précédente.

## Objectif du Projet
L'objectif est de suivre l'évolution des revenus et de la rentabilité. Ce rapport permet de comparer le **YTD** (Year-to-Date) avec le **PYTD** (Prior Year-to-Date) en se basant sur trois indicateurs : la **Quantité**, le **Gross Profit** (Marge Brute) et les **Ventes**.

## Outils Utilisés
* **Nettoyage des données :** Utilisation de **Power Query** pour structurer les fichiers de ventes.
* **Modélisation :** Création d'un modèle de données pour lier les tables (Produits, Géographie, Temps).
* **Calculs (DAX) :** Création de mesures pour calculer les variations (YTD vs PYTD) et le pourcentage de marge (**GP%**).
* **Visualisation :** * **Waterfall Chart** pour voir l'augmentation ou la baisse du profit par mois.
    * **Treemap** pour identifier les 10 pays les moins performants.
    * **Scatter Chart** pour la segmentation de la rentabilité par compte.

## Analyses Clés
* **Analyse Comparative :** Comparaison mensuelle du **Gross Profit** entre cette année et l'année dernière selon le type de produit (*Indoor, Landscape, Outdoor*).
* **Focus Performance :** Identification des pays et des produits les moins performants pour prendre des décisions correctives.
* **Segmentation :** Analyse de la rentabilité par client pour isoler les segments les plus profitables.
